Layouts & Colors

LAYOUT
In order to switch from the theme's default full width layout to the boxed one, uncomment the line: 

<!--
	
<link href="assets/css/boxed.css" rel="stylesheet">
	
-->
The uncommented line will be: 
<link href="assets/css/boxed.css" rel="stylesheet">

SKIN COLOR
In order to change the color skin, replace skin-blue.css in all of the html files with the skin color of your choice from assets/css/ folder (screenshot below).

Contact

The contact form is functional, but you will have to replace the current e-mail address (wowthemesnet@gmail.com) with your own. Open contact.php file and find the line 23 if you're using Notepad++ editor in order to do that. Don't forget to call the necessary script (contact.js) at the bottom of the document as in demo (contact.html)
